# README

### Overview
Learn How to Create MyFactorybot

### Environment
Ruby: 2.7.4
Rails: 6.1.4
PostgreSQL: 13.3

### Set Up


```
$ docker compose build
```

```
$ docker compose up -d
```

```
$ docker compose exec web bundle exec rails runner my_factory_bot.rb
```
